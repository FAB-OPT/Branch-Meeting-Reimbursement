จัดการรอบเบิก (CYCLE_MASTER)

* แก้ชื่อ/วันที่/เปิด-ปิดได้ แล้วกด “บันทึกรอบ”
cycleKey ชื่อรอบ เริ่ม จบ เปิด
วันที่ส่ง สาขา ชื่อผู้เบิก ยอด สถานะ

คู่มือเร็ว ✨

1) วาง URL Web App (ลงท้ายด้วย /exec)
2) ใส่รหัสสาขา (รองรับ 4–6 หลัก) หรือรหัสแอดมิน
3) ระบบจะดึง “ชื่อรอบเบิก” จาก CYCLE_MASTER
4) สาขาส่งได้เมื่อรอบ “เปิดใช้งาน” เท่านั้น

หมายเหตุ: การส่งรูปผ่าน JSONP มีข้อจำกัดความยาว URL — หน้าเว็บจะบีบอัดภาพให้เบาโดยอัตโนมัติ

ตั้งค่าแนะนำ ✅

  • ตั้ง BRANCH_MASTER!A เป็น Plain text
  • ใน CYCLE_MASTER แนะนำให้ใช้วันที่เป็น Date จริง หรือรูปแบบ dd/MM/yyyy
  • Deploy Web App: Execute as Me และ Access Anyone with the link
